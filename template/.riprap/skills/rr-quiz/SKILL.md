name: rr-quiz
description: Gives the user a quick quiz to test code comprehension.
allowed-tools: Read, Grep, Glob
---

Ask the user a multiple-choice question regarding the nuances of how the code works.  This question can include a short code snippet (1-10 lines).  If the user responds incorrectly, provide a detailed explanation of the subject.

# Question Format

Question.

**1.** Answer 1
**2.** Answer 2
**3.** Answer 3
**4.** Answer 4
